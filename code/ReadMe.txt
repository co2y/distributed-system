程序详细执行方式：
（1）环境：python 3.4.3，请确保是python3.4 而不是python2.7

（2）执行方法：
先执行python park.py [entrynum] [exitnum] [parkingnum]，启动停车场，例如：python park.py 3 3 3
再执行python test.py，启动测试脚本输入命令进行测试

（3）程序参数及测试命令说明：
执行python park.py 3 3 3时的三个参数依次代表入口数，出口数和车位数，可自行设置，例如也可以执行：python park.py 5 3 100
执行python test.py时的命令格式为[编号,IN/OUT]，命令请使用英文半角字符且对大小写敏感，例如：对于3个入口3个出口的情况来说，只接受1,IN 2,IN 3,IN 4,OUT 5,OUT 6,OUT这几种命令以及他们的组合。组合使用表示多个事件同时发生，用一个空格隔开，例如：1,IN 2,IN 3,IN
